# WarNet-Robot
🤖 La WarNet Robot, es la rama Cientifica de pupitres de la WarNet en dispositivos terrestres, centrada en la coordinación robótica y de enjambre en equipamiento de Tierra, para ser asumido en la coordinación con el resto de dispositivos robóticos de coordinación distribuida de las distintas fracciones WarNet.
